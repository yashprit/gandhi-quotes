#  gandhi-quotes [![NPM version][npm-image]][npm-url]

> show gandhi quotes


## Install

```sh
$ npm install --save gandhi-quotes
```


## Usage

```js
var gandhi-quotes = require('gandhi-quotes');

gandhi-quotes();
```

```sh
$ npm install --global gandhi-quotes
$ gandhi-quotes --p
#=> show with ascii image of gandhi
```
